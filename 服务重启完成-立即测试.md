## 🎯 功能说明

现在你可以使用以下功能：

### 1. 定义常量变量

```yaml
api_base_url:
  type: string
  source: constant
  value: "http://10.10.20.10:5000"

min_salary:
  type: number
  source: constant
  value: 15000
```

### 2. 自动可用（无需声明依赖）

```yaml
# ❌ 修改前（繁琐）
tech_users:
  source: api
  dependencies: [api_base_url]  # 必须声明
  api_config:
    endpoint: "{{api_base_url}}/api/users"

# ✅ 修改后（简洁）
tech_users:
  source: api
  # 无需 dependencies，常量自动可用
  api_config:
    endpoint: "{{api_base_url}}/api/users"
```

### 3. 类型自动保持

```yaml
min_salary:
  source: constant
  value: 15000  # 数字

query_api:
  source: api
  api_config:
    body:
      min_salary: "{{min_salary}}"  # → 15000 (数字，不是字符串)
```
