## 功能 1：智能类型保持

### 问题描述

**修复前**：
```yaml
body:
  department: "{{department_name}}"  # → "技术部" (string) ✅
  min_salary: "{{min_salary_value}}" # → "15000" (string) ❌ 错误！应该是数字
```

**修复后**：
```yaml
body:
  department: "{{department_name}}"      # → "技术部" (string) ✅
  min_salary: "{{min_salary_value}}"     # → 15000 (number) ✅ 正确！
  message: "最低：{{min_salary_value}}"  # → "最低：15000" (string) ✅
```

```python
def _interpolate_with_type_preservation(self, value: str) -> Any:
    """智能类型保持"""
    pure_var_pattern = r'^\s*\{\{([^}]+)\}\}\s*$'
    match = re.match(pure_var_pattern, value)
    
    if match:
        # 纯变量引用 - 保持类型
        var_expr = match.group(1).strip()
        if '|' not in var_expr:
            return self.variables[var_expr]  # 保持原始类型
    
    # 模板字符串 - 返回字符串
    return self.interpolate_string(value)
```

### 实施内容

#### 1. 更新枚举和模型
**文件**: `backend/app/core/models.py`

- ✅ 在 `VariableSource` 枚举中添加 `CONSTANT = "constant"`
- ✅ 在 `VariableMetadata` 中添加 `value: Optional[Any] = None` 字段

#### 2. 创建执行器
**文件**: `backend/app/executors/constant.py`（新建）

- ✅ 继承 `BaseVariableExecutor`
- ✅ `_execute_impl()` 方法直接返回 `metadata.value`
- ✅ 验证 value 字段存在
- ✅ 适当的错误处理

```python
class ConstantExecutor(BaseVariableExecutor):
    async def _execute_impl(self) -> Any:
        if self.metadata.value is None:
            raise ValueError(f"Constant variable must have 'value' field")
        return self.metadata.value
```

## 使用示例

### 智能类型保持示例

```python
# 定义变量
context.variables = {
    'department_name': '技术部',
    'min_salary': 15000,
    'is_active': True,
    'discount_rate': 0.85
}

# API body
body = {
    'department': '{{department_name}}',     # → "技术部" (str)
    'min_salary': '{{min_salary}}',          # → 15000 (int) ✅
    'active': '{{is_active}}',               # → True (bool) ✅
    'rate': '{{discount_rate}}',             # → 0.85 (float) ✅
    'message': '部门：{{department_name}}'    # → "部门：技术部" (str)
}

result = context.interpolate_dict(body)
# 数字、布尔类型正确保持！
```

### CONSTANT 变量使用示例

```yaml
# 定义常量
min_salary_standard:
  type: number
  source: constant
  description: "公司最低薪资标准"
  value: 15000

api_base_url:
  type: string
  source: constant
  description: "API基础地址"
  value: "http://10.10.20.10:5000"

vat_rate:
  type: number
  source: constant
  description: "增值税率"
  value: 0.13

# 在 API 调用中使用常量
user_query:
  type: object
  source: api
  dependencies: [min_salary_standard, api_base_url]
  api_config:
    endpoint: "{{api_base_url}}/api/users"
    method: POST
    body:
      min_salary: "{{min_salary_standard}}"  # → 15000 (number) ✅
      department: "技术部"
```
