# 变量元数据字段说明文档

## 基础字段说明

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| type | string | 是 | 变量数据类型 | "string", "number", "boolean", "array", "object" |
| source | string | 是 | 数据来源类型 | "user_input", "sql", "api", "ai_generation", "system" |
| required | boolean | 是 | 是否必须提供值 | true, false |
| description | string | 是 | 变量功能描述 | "报告标题", "用户档案信息" |
| default | any | 否 | 默认值（为空时使用） | "未命名报告", [], {}, 0 |
| dependencies | array | 否 | 依赖的其他变量名 | ["user_id"], ["project_id", "time_period"] |
| schema | object | 否 | 数据结构定义 | 见下方schema详细说明 |

## 数据源特定配置

### 用户输入变量 (source: "user_input")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| ui_config | object | 是 | 用户界面配置 |  |
| ui_config.input_type | string | 是 | 输入控件类型 | "text", "textarea", "number", "select", "checkbox", "date" |
| ui_config.placeholder | string | 否 | 输入框占位文本 | "请输入报告标题" |

### SQL数据库变量 (source: "sql")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| sql_config | object | 是 | SQL查询配置 |  |
| sql_config.connection | string | 是 | 数据库连接名称 | "user_db", "project_db" |
| sql_config.query | string | 是 | SQL查询语句 | "SELECT name FROM users WHERE id = {{user_id}}" |
| sql_config.parameters | array | 否 | 查询参数变量名 | ["user_id"], ["project_id", "start_date"] |
| sql_config.timeout | number | 否 | 查询超时时间(秒) | 10, 30 |

### API接口变量 (source: "api")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| api_config | object | 是 | API调用配置 |  |
| api_config.endpoint | string | 是 | API接口地址 | "https://api.weather.com/v1/current    " |
| api_config.method | string | 是 | HTTP方法 | "GET", "POST", "PUT", "DELETE" |
| api_config.headers | object | 否 | 请求头配置 | {"Authorization": "Bearer {{api_key}}"} |
| api_config.parameters | object | 否 | URL参数 | {"location": "{{city}}", "units": "metric"} |
| api_config.body | object | 否 | 请求体数据(POST/PUT) | {"symbols": "{{stock_symbols}}"} |
| api_config.response_mapping | object | 是 | 响应数据映射规则 | 见下方response_mapping说明 |
| api_config.cache_ttl | number | 否 | 缓存时间(秒) | 1800 (30分钟) |

### AI生成变量 (source: "ai_generation")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| ai_config | object | 是 | AI生成配置 |  |
| ai_config.model | string | 是 | AI模型名称 | "gpt-4", "gpt-3.5-turbo" |
| ai_config.prompt_template | string | 是 | 提示词模板 | "基于{{data}}生成分析报告" |
| ai_config.response_format | string | 否 | 期望响应格式 | "json_array", "json_object", "markdown" |
| ai_config.temperature | number | 否 | 生成随机性(0-1) | 0.3, 0.7 |

### 系统变量 (source: "system")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| system_config | object | 是 | 系统生成配置 |  |
| system_config.fields | object | 是 | 字段生成规则 | 见下方system_config说明 |

## 配置对象详细说明

### Schema 数据结构定义

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| type | string | 是 | 数据类型 | "object", "array", "string", "number" |
| properties | object | 对象必填 | 对象属性定义 | {"name": {"type": "string"}} |
| items | object | 数组必填 | 数组元素定义 | {"type": "object", "properties": {...}} |
| format | string | 否 | 数据格式 | "date", "email", "url" |
| enum | array | 否 | 枚举值列表 | ["高", "中", "低"] |

### API Response Mapping 响应映射

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| {field_name} | string | 对象映射 | 字段映射路径 | "data.current.temp" |
| extract_array | string | 数组映射 | 数组提取路径 | "data.stocks" |
| item_mapping | object | 数组映射 | 数组元素字段映射 | {"symbol": "symbol", "price": "price"} |

### System Config 系统配置

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| generator | string | 是 | 生成器类型 | "datetime", "uuid" |
| format | string | 否 | 输出格式 | "YYYY-MM-DD HH:mm:ss" |
| value | any | 否 | 固定值 | "1.0" |

## 数据类型说明

### type 字段可选值

| 类型值 | 描述 | 使用场景 |
|--------|------|----------|
| "string" | 字符串类型 | 文本内容、名称、描述等 |
| "number" | 数字类型 | 数量、百分比、价格等 |
| "boolean" | 布尔类型 | 开关状态、是否标记等 |
| "array" | 数组类型 | 列表数据、多项选择等 |
| "object" | 对象类型 | 复杂数据结构、嵌套数据等 |

### source 字段可选值

| 来源值 | 描述 | 使用场景 |
|--------|------|----------|
| "user_input" | 用户输入 | 需要用户填写或选择的数据 |
| "sql" | 数据库查询 | 从数据库中获取的数据 |
| "api" | 接口调用 | 从外部API获取的数据 |
| "ai_generation" | AI生成 | 通过AI模型生成的内容 |
| "system" | 系统生成 | 系统自动生成的数据（时间、ID等） |

## 完整配置示例

### 用户输入变量示例

```yaml
report_title:
  type: "string"
  source: "user_input"
  required: true
  description: "报告标题"
  default: "未命名报告"
  ui_config:
    input_type: "text"
    placeholder: "请输入报告标题"



SQL数据库变量示例
yaml
user_profile:
  type: "object"
  source: "sql"
  required: false
  default: {}
  description: "用户档案信息"
  dependencies: ["user_id"]
  sql_config:
    connection: "user_db"
    query: "SELECT name, email, department, role FROM users WHERE id = {{user_id}}"
    parameters: ["user_id"]
    timeout: 10
  schema:
    type: "object"
    properties:
      name: {type: "string"}
      email: {type: "string"}
      department: {type: "string"}
      role: {type: "string"}


API接口变量示例
yaml
weather_data:
  type: "object"
  source: "api"
  required: false
  default: {}
  description: "天气数据"
  dependencies: ["location"]
  api_config:
    endpoint: "https://api.weather.com/v1/current    "
    method: "GET"
    headers:
      Authorization: "Bearer {{weather_api_key}}"
    parameters:
      location: "{{location}}"
    response_mapping:
      temperature: "data.current.temp"
      condition: "data.current.condition"
      humidity: "data.current.humidity"
    cache_ttl: 1800
  schema:
    type: "object"
    properties:
      temperature: {type: "number"}
      condition: {type: "string"}
      humidity: {type: "number"}
AI生成变量示例
yaml
risk_analysis:
  type: "array"
  source: "ai_generation"
  required: false
  default: []
  description: "风险分析结果"
  dependencies: ["project_tasks", "project_info"]
  ai_config:
    model: "gpt-4"
    prompt_template: |
      基于项目任务和项目信息进行风险分析：
      项目信息：{{project_info}}
      任务列表：{{project_tasks}}
      
      请识别3-5个主要风险，每个风险包含：
      - 风险描述
      - 影响等级（高/中/低）
      - 发生概率（高/中/低）
      - 缓解建议
      
      以JSON数组格式返回。
    response_format: "json_array"
    temperature: 0.3
  schema:
    type: "array"
    items:
      type: "object"
      properties:
        risk_description: {type: "string"}
        impact_level: {type: "string", enum: ["高", "中", "低"]}
        probability: {type: "string", enum: ["高", "中", "低"]}
        suggestion: {type: "string"}
系统变量示例

generation_info:
  type: "object"
  source: "system"
  required: true
  description: "生成信息"
  system_config:
    fields:
      timestamp:
        generator: "datetime"
        format: "YYYY-MM-DD HH:mm:ss"

        generator: "uuid"
      version:
        value: "1.0"
  schema:
    type: "object"
    properties:
      timestamp: {type: "string"}
      report_id: {type: "string"}
      version: {type: "string"}
