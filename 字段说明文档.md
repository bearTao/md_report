# 变量元数据字段说明文档

## 基础字段说明

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| type | string | 是 | 变量数据类型 | "string", "number", "boolean", "array", "object" |
| source | string | 是 | 数据来源类型 | "user_input", "sql", "api", "ai_generation", "system" |
| required | boolean | 是 | 是否必须提供值 | true, false |
| description | string | 是 | 变量功能描述 | "报告标题", "用户档案信息" |
| default | any | 否 | 默认值（为空时使用） | "未命名报告", [], {}, 0 |
| dependencies | array | 否 | 依赖的其他变量名 | ["user_id"], ["project_id", "time_period"] |
| schema | object | 否 | 数据结构定义 | 见下方schema详细说明 |

## 数据源特定配置

### 用户输入变量 (source: "user_input")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| ui_config | object | 是 | 用户界面配置 |  |
| ui_config.input_type | string | 是 | 输入控件类型 | "text", "textarea", "number", "select", "checkbox", "date" |
| ui_config.placeholder | string | 否 | 输入框占位文本 | "请输入报告标题" |

### SQL数据库变量 (source: "sql")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| sql_config | object | 是 | SQL查询配置 |  |
| sql_config.connection | string | 是 | 数据库连接名称 | "user_db", "project_db" |
| sql_config.query | string | 是 | SQL查询语句 | "SELECT name FROM users WHERE id = {{user_id}}" |
| sql_config.parameters | array | 否 | 查询参数变量名 | ["user_id"], ["project_id", "start_date"] |
| sql_config.timeout | number | 否 | 查询超时时间(秒) | 10, 30 |
| sql_config.result_mode | string | 否 | 结果返回模式 | "first_row", "all_rows", "first_value", "first_column", "auto" (默认) |

**result_mode 说明：**
- `first_row`: 返回第一行作为对象 `{col1: val1, col2: val2}`
- `all_rows`: 返回所有行作为数组 `[{row1}, {row2}, ...]`
- `first_value`: 返回第一行第一列的值（标量）
- `first_column`: 返回第一列的所有值 `[val1, val2, ...]`
- `auto`: 根据 `type` 自动判断（默认行为）
  - `type="object"`: 单行返回对象，多行返回数组
  - `type="array"`: 返回所有行
  - `type="string/number/boolean"`: 返回第一行第一列

| api_config.response_mapping | object | 是 | 响应数据映射规则 | 见下方response_mapping说明 |

### 图片变量 (source: "image")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| image_config | object | 是 | 图片获取配置 |  |
| image_config.endpoint | string | 是 | 图片URL或API端点 | "https://api.example.com/image.jpg" |
| image_config.method | string | 否 | HTTP方法 | "GET" (默认) |
| image_config.headers | object | 否 | 请求头 | {"Authorization": "Bearer {{token}}"} |
| image_config.parameters | object | 否 | URL参数 | {"size": "large", "quality": "80"} |
| image_config.output_format | string | 否 | 输出格式 | "url" (默认), "base64", "bytes" |
| image_config.multiple | boolean | 否 | 是否获取多张图片 | false (默认), true |
| image_config.timeout | number | 否 | 请求超时时间(秒) | 30 (默认) |

**output_format 说明：**
- `url`: 返回原始URL（推荐，速度快，文件小）
- `base64`: 返回Base64编码（适合离线报告）
- `bytes`: 返回原始字节数据（适合二次处理）

**返回数据格式（单张图片）：**
```json
{
  "data": "https://example.com/image.jpg",  // 或base64字符串
  "url": "https://example.com/image.jpg",
  "mime_type": "image/jpeg",
  "size": 12345,
  "markdown": "![图片](https://...)"
}
```

**返回数据格式（多张图片）：**
```json
[
  {"data": "...", "url": "...", "mime_type": "...", "size": ..., "markdown": "..."},
  {"data": "...", "url": "...", "mime_type": "...", "size": ..., "markdown": "..."}
]
```

### Vision AI变量 (source: "vision_ai")

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| vision_ai_config | object | 是 | Vision AI配置 |  |
| vision_ai_config.model | string | 是 | 视觉模型名称 | "gpt-4o", "gpt-4-vision-preview" |
| vision_ai_config.image_source | string | 是 | 图片变量名称 | "cover_image", "site_photos" |
| vision_ai_config.prompt_template | string | 是 | 提示词模板 | "请分析这张图片..." |
| vision_ai_config.temperature | number | 否 | 生成随机性(0-1) | 0.7 (默认) |
| vision_ai_config.max_tokens | number | 否 | 最大输出长度 | 500 (默认) |
| vision_ai_config.system_prompt | string | 否 | 系统提示词 | "你是一位图像分析专家" |

**注意事项：**
1. Vision AI变量必须依赖一个图片变量（通过`dependencies`和`image_source`指定）
2. 支持分析单张或多张图片
3. 需要配置OpenAI API Key才能使用
4. 返回文本分析结果

## 配置对象详细说明

### Schema 数据结构定义

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| type | string | 是 | 数据类型 | "object", "array", "string", "number" |
| properties | object | 对象必填 | 对象属性定义 | {"name": {"type": "string"}} |
| items | object | 数组必填 | 数组元素定义 | {"type": "object", "properties": {...}} |
| format | string | 否 | 数据格式 | "date", "email", "url" |
| enum | array | 否 | 枚举值列表 | ["高", "中", "低"] |

### API Response Mapping 响应映射

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| {field_name} | string | 对象映射 | 字段映射路径 | "data.current.temp" |
| extract_array | string | 数组映射 | 数组提取路径 | "data.stocks" |
| item_mapping | object | 数组映射 | 数组元素字段映射 | {"symbol": "symbol", "price": "price"} |

### System Config 系统配置

| 字段名 | 类型 | 必填 | 描述 | 示例值 |
|--------|------|------|------|---------|
| generator | string | 是 | 生成器类型 | "datetime", "uuid" |
| format | string | 否 | 输出格式 | "YYYY-MM-DD HH:mm:ss" |
| value | any | 否 | 固定值 | "1.0" |

## 数据类型说明

### type 字段可选值

| 类型值 | 描述 | 使用场景 |
|--------|------|----------|
| "string" | 字符串类型 | 文本内容、名称、描述等 |
| "number" | 数字类型 | 数量、百分比、价格等 |
| "boolean" | 布尔类型 | 开关状态、是否标记等 |
| "array" | 数组类型 | 列表数据、多项选择等 |
| "object" | 对象类型 | 复杂数据结构、嵌套数据等 |

### source 字段可选值

| 来源值 | 描述 | 使用场景 |
|--------|------|----------|
| "user_input" | 用户输入 | 需要用户填写或选择的数据 |
| "sql" | 数据库查询 | 从数据库中获取的数据 |
| "api" | 接口调用 | 从外部API获取的数据 |
| "ai_generation" | AI生成 | 通过AI模型生成的内容 |
| "image" | 图片获取 | 从HTTP API获取图片数据 |
| "vision_ai" | 视觉AI分析 | 使用视觉大模型分析图片内容 |
| "system" | 系统生成 | 系统自动生成的数据（时间、ID等） |

### SQL数据库变量示例

```yaml
# 示例1：获取单个用户档案（返回对象）
user_profile:
  type: "object"
  source: "sql"
  required: false
  default: {}
  description: "用户档案信息"
  dependencies: ["user_id"]
  sql_config:
    connection: "user_db"
    query: "SELECT name, email, department, role FROM users WHERE id = {{user_id}}"
    parameters: ["user_id"]
    timeout: 10
    result_mode: "first_row"  # 明确返回第一行作为对象
  schema:
    type: "object"
    properties:
      name: {type: "string"}
      email: {type: "string"}
      department: {type: "string"}
      role: {type: "string"}

# 示例2：获取用户列表（返回数组）
team_members:
  type: "array"
  source: "sql"
  required: false
  default: []
  description: "团队成员列表"
  dependencies: ["department_id"]
  sql_config:
    connection: "user_db"
    query: "SELECT name, email, role FROM users WHERE department_id = {{department_id}}"
    parameters: ["department_id"]
    timeout: 10
    result_mode: "all_rows"  # 返回所有行
  schema:
    type: "array"
    items:
      type: "object"
      properties:
        name: {type: "string"}
        email: {type: "string"}
        role: {type: "string"}

# 示例3：获取总计数（返回标量）
total_sales:
  type: "number"
  source: "sql"
  required: false
  default: 0
  description: "总销售额"
  dependencies: ["start_date", "end_date"]
  sql_config:
    connection: "analytics_db"
    query: "SELECT SUM(amount) FROM sales WHERE date BETWEEN {{start_date}} AND {{end_date}}"
    parameters: ["start_date", "end_date"]
    timeout: 10
    result_mode: "first_value"  # 返回第一行第一列的值

# 示例4：获取用户ID列表（返回单列数组）
user_ids:
  type: "array"
  source: "sql"
  required: false
  default: []
  description: "活跃用户ID列表"
  dependencies: []
  sql_config:
    connection: "user_db"
    query: "SELECT id FROM users WHERE status = 'active'"
    parameters: []
    timeout: 10
    result_mode: "first_column"  # 返回第一列的所有值
  schema:
    type: "array"
    items:
      type: "number"
```
