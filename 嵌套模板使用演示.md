# 嵌套模板使用演示

## 📖 演示场景

创建一个月度报告，包含可复用的销售总结子模板。

---

## 步骤1：创建子模板（销售总结）

### 访问
http://10.10.20.10:5173/templates → 点击"新建模板"

### 填写内容

**基本信息**:
- 名称：`销售总结模板`
- 描述：`可复用的销售数据总结组件`

**模板内容**:
```jinja2
## 销售数据总结

- 销售负责人：{{sales_owner}}
- 本月销售额：{{sales_amount}}元
- 订单数量：{{order_count}}笔
```

**变量元数据**:
```yaml
sales_owner:
  type: string
  source: user_input
  required: true
  description: 销售负责人
  ui_config:
    input_type: text
    placeholder: 请输入销售负责人姓名

sales_amount:
  type: number
  source: user_input
  required: true
  description: 销售额
  ui_config:
    input_type: number
    placeholder: 请输入销售额

order_count:
  type: number
  source: user_input
  required: true
  description: 订单数量
  ui_config:
    input_type: number
    placeholder: 请输入订单数量
```

点击"保存模板"，记住模板ID（如 `tpl_abc123`）

---

## 步骤2：创建主模板（月度报告）

### 新建模板

**基本信息**:
- 名称：`月度报告模板`
- 描述：`包含销售总结的月度报告`

**模板内容**:
```jinja2
# {{month}}月度报告

报告负责人：{{report_owner}}

---

{% include "tpl_abc123" %}

---

## 总结
本月工作顺利完成。
```

**注意**: 将 `tpl_abc123` 替换为步骤1中创建的子模板ID

**变量元数据**:
```yaml
month:
  type: string
  source: user_input
  required: true
  description: 月份
  ui_config:
    input_type: text
    placeholder: 请输入月份（如：10）

report_owner:
  type: string
  source: user_input
  required: true
  description: 报告负责人
  ui_config:
    input_type: text
    placeholder: 请输入报告负责人姓名
```

点击"保存模板"

---

## 步骤3：生成报告

### 访问生成报告页面
http://10.10.20.10:5173/generate

### 选择模板
选择刚才创建的"月度报告模板"

### 填写数据（分层表单）

您会看到两个Card section：

**主模板: 月度报告模板**
- 月份：`10`
- 报告负责人：`张三`

**子模板: 销售总结模板 (ID: tpl_abc123)**
- 销售负责人：`李四`
- 销售额：`500000`
- 订单数量：`120`

### 点击"开始生成"

---

## 步骤4：查看结果

生成的报告内容：

```markdown
# 10月度报告

报告负责人：张三

---

## 销售数据总结

- 销售负责人：李四
- 本月销售额：500000元
- 订单数量：120笔

---

## 总结
本月工作顺利完成。
```

### 验证变量隔离

注意：
- 主模板的"报告负责人"是"张三" ✅
- 子模板的"销售负责人"是"李四" ✅
- 两个变量完全独立，没有相互影响 🎉

---

## 💡 高级用法

### 同名变量测试

您可以在主模板和子模板中都定义 `title` 变量，测试变量隔离：

**主模板**:
```jinja2
# {{title}}

{% include "sub_template" %}
```

**子模板**:
```jinja2
## {{title}}
```

生成时：
- 主模板 title: `主标题`
- 子模板 title: `子标题`

结果：
```markdown
# 主标题

## 子标题
```

## 🐛 使用调试功能测试

### 在创建模板前先调试

1. 点击左侧菜单"调试测试"
2. 在调试页面编写模板和元数据
3. 测试渲染效果
4. 确认无误后，复制到模板编辑页面保存

### 从模板编辑页快速调测

1. 打开模板编辑页
2. 点击"调测模板"按钮
3. 自动跳转到调试页面（内容已填充）
4. 填写用户输入并执行调试
