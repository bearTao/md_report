## 🚀 功能演示

### 使用场景1: 产品质量检测

```yaml
# 模板变量配置
product_photo:
  type: image
  source: image
  dependencies: [product_id]
  image_config:
    endpoint: "https://api.example.com/products/{{product_id}}/photo"
    output_format: base64
    
quality_check:
  type: string
  source: vision_ai
  dependencies: [product_photo]
  vision_ai_config:
    model: gpt-4o
    image_source: product_photo
    prompt_template: "请检查这个产品的外观质量"
```

### 使用场景2: 现场安全检查

```yaml
# 批量图片 + AI分析
site_photos:
  type: array
  source: image
  image_config:
    endpoint: "https://api.example.com/sites/{{site_id}}/photos"
    multiple: true  # 批量获取
    
safety_report:
  type: string
  source: vision_ai
  vision_ai_config:
    model: gpt-4o
    image_source: site_photos  # 引用批量图片
    prompt_template: "分析这些现场照片，识别安全隐患"
```
