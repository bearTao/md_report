## 🔧 主要修改

### 1. 数据库Schema更新

在 `app/models/db_models.py` 中新增:

```python
class DBEngineType(str, enum.Enum):
    """Database engine types"""
    POSTGRESQL = "postgresql"
    MYSQL = "mysql"
    SQLSERVER = "sqlserver"
    ORACLE = "oracle"

class DBConnection(Base):
    """Database connection configurations"""
    __tablename__ = "db_connections"
    
    id = Column(String(50), primary_key=True)
    name = Column(String(200), nullable=False, unique=True)
    engine = Column(SQLEnum(DBEngineType), nullable=False)
    host = Column(String(500), nullable=False)
    port = Column(Integer, nullable=False)
    database = Column(String(200), nullable=False)
    username = Column(String(200), nullable=False)
    password_ciphertext = Column(Text, nullable=False)
    options_json = Column(JSON)
    is_active = Column(String(10), default="true")
    created_at = Column(DateTime(timezone=True), server_default=func.now())
    updated_at = Column(DateTime(timezone=True), server_default=func.now(), onupdate=func.now())
```

### 2. 报告生成服务集成WebSocket

在 `app/api/reports.py` 的 `execute_report_generation` 函数中:

- ✅ 任务开始时推送 `task_started` 事件
- ✅ 变量开始执行时推送 `variable_started` 事件
- ✅ 变量完成时推送 `variable_completed/failed` 事件
- ✅ 任务完成/失败时推送 `task_completed/failed` 事件

### 3. 路由注册

在 `app/main.py` 中添加:

```python
from app.api import websocket, db_connections

app.include_router(websocket.router)
app.include_router(db_connections.router)
```

## 🚀 使用示例

### 1. 连接WebSocket接收实时更新

```javascript
const ws = new WebSocket('ws://localhost:8000/ws/report-generation/task_123');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Event:', data.type, data);
  
  switch(data.type) {
    case 'task_started':
      console.log('任务开始:', data.started_at);
      break;
    case 'variable_completed':
      console.log('变量完成:', data.variable_name);
      break;
    case 'task_completed':
      console.log('任务完成, 报告ID:', data.report_id);
      break;
  }
};
```

### 2. 查询报告历史

```bash
# 获取所有成功的报告
curl "http://localhost:8000/api/reports/?status=success&page=1&page_size=10"

# 获取指定模板的报告
curl "http://localhost:8000/api/reports/?template_id=tpl_xxx"
```

### 3. 管理数据库连接

```bash
# 创建连接
curl -X POST http://localhost:8000/api/config/db-connections/ \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Production MySQL",
    "engine": "mysql",
    "host": "db.example.com",
    "port": 3306,
    "database": "prod_db",
    "username": "admin",
    "password": "secret"
  }'

# 测试连接
curl -X POST http://localhost:8000/api/config/db-connections/{id}/test
```
