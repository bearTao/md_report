# 模板测试模块

模板系统的测试，包括模板渲染引擎、嵌套模板和模板功能验证。

## 📦 测试文件说明

- **test_renderer.py** - 模板渲染器测试
- **test_template_nesting.py** - 模板嵌套和调试功能测试

## 🎯 测试覆盖范围

### 模板渲染器
- Jinja2 模板渲染
- 变量替换和注入
- 模板语法验证
- 渲染错误处理

### 模板嵌套
- 子模板引用和包含
- 嵌套层级处理
- 循环引用检测
- 模板继承

### 调试功能
- 模板调试信息
- 变量追踪
- 渲染过程日志

## 🚀 运行测试

```bash
# 运行所有模板测试
pytest tests/template/

# 运行特定测试文件
pytest tests/template/test_template_nesting.py

# 查看详细输出
pytest tests/template/ -v
```

## 📝 注意事项

- 模板测试需要数据库会话来创建测试模板
- 嵌套测试会创建临时模板数据
- 测试后会自动清理创建的测试数据
- 注意模板语法的特殊字符转义
