## 🐛 发现的问题

### 问题1: 后台任务数据库Session问题 🔴 Critical

**症状**:
- 报告生成任务创建成功
- 任务状态一直停留在`running`
- 变量列表为空
- 60秒超时无响应

**根本原因**:
```python
# app/api/reports.py:469
background_tasks.add_task(
    execute_report_generation,
    ...
    db_session=db,  # ⚠️ Request-scoped session
    ...
)
```

FastAPI的数据库session是request-scoped的。当HTTP请求返回后，session就关闭了，但后台任务仍在运行，导致无法访问数据库。

**影响范围**:
- 所有报告生成功能
- 变量执行记录无法保存
- WebSocket事件无法发送

**修复方案**:
```python
# 方案1：在后台任务中创建新session（推荐）
async def execute_report_generation(
    task_id: str,
    template_id: str,
    template_content: str,
    metadata: Dict[str, Any],
    user_inputs: Dict[str, Any],
    # 移除 db_session 参数
    openai_api_key: Optional[str] = None,
    openai_api_base: Optional[str] = None
):
    # 创建新session
    from app.database import SessionLocal
    db = SessionLocal()
    try:
        # 原有逻辑...
        task = db.query(GenerationTask).filter(
            GenerationTask.id == task_id
        ).first()
        # ...
    finally:
        db.close()

# generate_report函数调用
background_tasks.add_task(
    execute_report_generation,
    task_id=task_id,
    template_id=request.template_id,
    template_content=template.template_content,
    metadata=template.metadata_json,
    user_inputs=request.inputs,
    # 移除 db_session=db
    openai_api_key=openai_api_key,
    openai_api_base=openai_api_base
)
```

**预计修复时间**: 2-4小时
