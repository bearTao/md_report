## 日志级别说明

### DEBUG 级别
- 最详细的信息，用于开发和深度调试
- 内容：变量列表、模板大小、模板内容预览、会话关闭等

### INFO 级别
- 关键流程信息，适合生产环境监控
- 内容：渲染开始/结束、耗时、输出大小、连接注册、重试成功等

### WARNING 级别
- 警告信息，不影响功能但需要注意
- 内容：API 重试、配置降级等

### ERROR 级别
- 错误信息，功能失败或异常
- 内容：渲染失败、变量执行失败、数据库错误等
- 使用 `exc_info=True` 参数自动记录完整堆栈
