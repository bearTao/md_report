# P1功能快速开始指南

## 🚀 启动服务

### 1. 启动后端服务

```bash
cd /data/tao/code/xuqiu/backend
python3 -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 启动前端服务

```bash
cd /data/tao/code/xuqiu/frontend
npm run dev
```

### 3. 访问应用

- 前端: http://localhost:5173
- 后端API文档: http://localhost:8000/docs

## 🐛 常见问题

### Q1: 取消任务后仍在执行？
**A**: 取消操作依赖于检查点。对于长时间运行的SQL/API调用，可能需要等待当前操作完成。

### Q2: 日志不显示？
**A**: 确认：
1. 数据库连接正常
2. `execution_logs`表已创建
3. 后端日志服务正常工作
4. 任务ID正确

### Q3: 模板校验报错"Template not found"？
**A**: 只能校验已保存的模板。新建模板需要先保存后再校验。

### Q4: 重试变量后报告仍未生成？
**A**: 检查：
1. 所有变量是否都成功
2. 后端日志是否有错误
3. 模板渲染是否正常
